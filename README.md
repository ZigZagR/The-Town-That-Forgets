# The Town That Forgets
## Uma aventura narrativa de texto em um universo lovecraftiano

## Requisitos: 
### g++ e windows.
## Para verificar instalação do g++:
```bash 
g++ --version
```
## Em caso de falha:
### Download g++ pelo https://www.msys2.org/ usando o guia deles.

## Clone o projeto em sua pasta de preferência:
```bash
git clone https://github.com/ZigZagR/The-Town-That-Forgets
```
## Como compilar: Abra o terminal dentro da pasta do projeto e:
```bash
g++ rpg.cpp -o rpg.exe
```
